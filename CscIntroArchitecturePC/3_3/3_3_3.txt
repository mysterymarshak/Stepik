Вопрос:
Отметьте верные утверждения

Ответ:
Для операций ввода/вывода могут использоваться прерывания
Указатель на обработчик системных вызовов можно найти в таблице векторов прерываний
Исключения — это особый класс прерываний, связанный с исключительными ситуациями, возникающими в процессе работы процессора
Прерывание таймера используется для управления многозадачностью
Программные прерывания — основа интерфейса операционной системы для приложений
